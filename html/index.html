Über Mich

Mein Name ist Oliver Morawski, ich bin 17 Jahre alt und leidenschaftlicher Entwickler. Zu meinen umgesetzten Projekten gehören der Pycord Discord Bot sowie eine eigene API-Website, auf der ich maßgeschneiderte APIs anbiete. Ich beherrsche grundlegend JavaScript, HTML, CSS, Kotlin, Java, PHP und SQLite. Fortgeschrittene Kenntnisse habe ich in Python, inklusive der Anwendung von PyTorch im Bereich Künstliche Intelligenz. Im Backend arbeite ich mit Tools wie Nginx, und Erfahrungen habe ich mit Frameworks wie Flask und Django gesammelt. Zudem nutze ich DevOps-Tools wie Docker, Bash und Azure. Mit umfangreichen Kenntnissen in Git und einer langen Nutzungsdauer von Linux (Ubuntu, Arch, Ubuntu Server) freue ich mich darauf, meine Entwicklungserfahrungen weiter zu vertiefen Vielen Dank für Ihr Interesse an meiner Reise durch die Welt der Softwareentwicklung!

Aniworld Discord RPC

Diese Discord RPC hat das Ziel, mithilfe von Webscraping den aktuell angeschauten Anime zu identifizieren und diesen dann als Discord-Aktivität anzuzeigen. Mit diesem Projekt möchte ich mein Verständnis für Webscraping vertiefen und Nutzern, die diese Seite verwenden, die Möglichkeit bieten, anderen mitzuteilen, welchen Anime sie gerade schauen. Darüber hinaus können möglicherweise noch zusätzliche Informationen angezeigt werden.

Osu Skin Api

Diese API hat den Zweck, das Herunterladen von osu!-Skins einfacher und schneller zu gestalten. Als Endpunkt dieser API fungiert eine Datenbank, die IDs, Namen und funktionierende Download-Links bereithält. Dieses Projekt möchte ich umsetzen, da ich bisher Flask kaum genutzt habe, um eine API zu erstellen. Durch dieses Projekt beabsichtige ich, mein Wissen in diesem Bereich zu vertiefen. Normalerweise verwende ich für kleine und einfache APIs JavaScript und binde sie dann über Requests in meine Projekte ein.

Automatische Dockerfile Creator

Ich plane die Entwicklung eines Dockerfile-Creators zur Automatisierung. Mit diesem Tool genügt die einfache Angabe des Ordners, für den ein Dockerfile erstellt werden soll, und der Rest geschieht automatisch. Das Programm durchsucht eigenständig die Ordner und Dateinamen, um automatisch ein Dockerfile zu generieren.
